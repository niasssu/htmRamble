basic design guide

background color is #ace. text color is #114. text divs should have a background brighter than #999, preferrably #fff or #eee.

border radius should be (at most) 1rem in most cases. borders should blend and be as thin as possible, if visible at all.

font is 10pt, default monospace. text should be all lowercase, but still use proper punctuation.

hyperlinks go in () brackets, use [] for parentheses.
